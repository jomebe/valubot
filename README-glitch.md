# VALUBOT - Discord Bot

VALUBOT은 발로란트 게임과 관련된 다양한 기능을 제공하는 Discord 봇입니다.

## 기능
- 음성 채널 관리
- 발로란트 전적 조회
- 출석체크 시스템
- TTS (Text-to-Speech)
- 음악 재생
- 선착순 시스템

## Glitch에서 실행하기

1. `.env` 파일에 환경변수 설정
2. 봇이 자동으로 시작됩니다
3. 슬립 방지를 위해 주기적으로 활동 로그가 출력됩니다

## 슬립 방지

- 10분마다 자동으로 콘솔 로그 출력
- 웹 서버가 `/` 와 `/keep-alive` 엔드포인트 제공
- UptimeRobot 등을 사용하여 5분마다 핑 가능
